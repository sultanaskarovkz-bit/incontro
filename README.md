# INCONTRO - Алексей Мицинский

Продающий лендинг психолога-психотерапевта Алексея Мицинского (бренд **INCONTRO**).
Индивидуальная психотерапия: тревога, выгорание, отношения, глубокая трансформация.

## Стек

Статический сайт без сборки - чистые **HTML / CSS / JS**.
Открывается двойным кликом по `index.html` или с любого статического хостинга.

## Структура

```
index.html      - главная страница (лендинг)
styles.css      - дизайн-система и адаптив
script.js       - интерактив: формы, слайдеры, дипломы, видео, модалка
privacy.html    - политика конфиденциальности
oferta.html     - публичная оферта
assets/
  img/          - фотографии, иконки, превью
  certs/        - дипломы и сертификаты (PDF)
```

## Возможности

- Премиальный адаптивный дизайн (десктоп, ноутбук, iPhone, Samsung)
- Формы заявки → WhatsApp/Telegram + экран «Спасибо»
- Видео-секция, Instagram-лента, мини-галерея, фильтруемые дипломы (49 шт.)
- SEO: Open Graph, Twitter Card, JSON-LD, favicon
- Заготовки под Meta Pixel / GA4 / Яндекс.Метрику (см. `<head>` в `index.html`)

## Деплой

Любой статический хостинг: GitHub Pages, Netlify, Vercel, Cloudflare Pages
или обычный веб-сервер. Сборка не требуется.
